# 🎬 Movie Ticket Booking System

A simple **Python Console-Based Movie Ticket Booking Application** that allows users to book movie tickets, purchase food combos, cancel bookings, and check seat availability.

This project demonstrates the concepts of **Object-Oriented Programming (OOP)**, dictionaries, loops, conditional statements, and user input handling in Python.

---

## 📌 Project Overview

The Movie Ticket Booking System enables users to:

- 🎟️ Book movie tickets
- 🍿 Purchase food combos
- ❌ Cancel booked tickets
- 🎬 View available movies
- 💰 Calculate the total booking amount automatically

The application is menu-driven and runs entirely in the terminal.
